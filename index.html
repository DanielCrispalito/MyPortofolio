Dethax

Description :


Dethax is a web-based application for detecting fake news in Indonesian using machine learning. The system combines modern NLP preprocessing with classical ML models to provide accurate classification of news articles.

  • Text Preprocessing: Tokenization, punctuation & stopwords removal, lemmatization.
  • Vectorization: TF-IDF feature extraction using TfidfVectorizer.
  • Imbalanced Data Handling: Applied SMOTE, SMOTEENN, and RandomUnderSampler.
  • Model Training: Logistic Regression, Naive Bayes, SVC, XGBoost, and Stacking Ensemble.
  • Evaluation: Accuracy, precision, recall, f1-score, and confusion matrix visualization.

Built with React.js, Flask, and scikit-learn, Dethax delivers a responsive web experience and achieves an accuracy of 83.15% with the Stacking + SMOTE model.
